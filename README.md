## 📍 Learning Scope (Completed)

### 🧠 Variables and Data Types

Variables are used to store values in memory.

Common data types:

- Integer → whole numbers  
- Float → decimal values  
- String → textual data  
- Boolean → logical values (True / False)

**Key Points to Remember**

- Python uses dynamic typing  
- Variable names should be meaningful and readable  
- Follow `snake_case` naming convention  
- Avoid overwriting built-in function names  

---

### ⌨️ User Input and Type Conversion

User input is taken using the `input()` function.

Important behaviour:

- Input is always received as a string  
- Numeric operations require type conversion  

Common conversion functions:

- `int()` → convert to integer  
- `float()` → convert to decimal  
- `str()` → convert to string  

---

### ➗ Operators

#### Arithmetic Operators

- Addition `+`  
- Subtraction `-`  
- Multiplication `*`  
- Division `/`  
- Modulus `%` → gives remainder  
- Exponent `**` → power calculation  
- Floor Division `//` → removes decimal part  

#### Comparison Operators

- Equal to `==`  
- Not equal to `!=`  
- Greater than `>`  
- Less than `<`  
- Greater or equal `>=`  
- Less or equal `<=`  

#### Logical Operators

- `and` → all conditions must be true  
- `or` → at least one condition must be true  
- `not` → reverses condition  

---

### 🔀 Conditional Statements

Conditional statements control decision making in programs.

Structure:

- `if` → executes when condition is true  
- `elif` → checks additional conditions  
- `else` → executes when all conditions are false  

**Important Concepts**

- Conditions return Boolean values  
- Indentation defines execution blocks  
- Program flow moves top to bottom  

---

### 🔁 Loops

Loops allow repeated execution of code.

#### For Loop

- Used when iteration count is known  
- Works with `range()`  

#### While Loop

- Used when iteration count is unknown  
- Condition must change to avoid infinite loops  

---

### 📚 Lists

Lists store multiple values in an ordered structure.

Properties:

- Mutable (modifiable)  
- Indexed (starts from 0)  
- Allows duplicate values  

Common operations:

- `append()` → add item  
- `remove()` → delete item  
- `len()` → list size  
- Iteration using loop  

---

### 🗂 Dictionaries

Dictionaries store data in key–value format.

Properties:

- Mutable  
- Keys must be unique  
- Values are accessed using keys  

Common operations:

- Add new key  
- Update value  
- Loop through dictionary  

---

### ⚙️ Functions

Functions are reusable blocks of logic.

Benefits:

- Improves readability  
- Avoids repetition  
- Enables modular programming  

Concepts learned:

- Function definition using `def`  
- Parameters and arguments  
- Returning values using `return`
